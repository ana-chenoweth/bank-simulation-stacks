Detailed Description

template<typename T>
class Cola< T >

Permite utilizar colas de datos genéricos. y ofrece un conjunto de operaciones básicas: agregar, eliminar, obtener el tamano de la cola, obtener el valor del fondo, obtener el valor del frente, imprimir, vaciar, etc.
